# Infopunks Launch Style

Never feel like SaaS.

Always feel like discovering classified intelligence.

Use:

- Cinematic typography
- Terminal animations
- Dark interfaces
- Signal extraction
- Graph visualizations
- Receipt evidence
- Minimal text
- Metaphor-first motion

Avoid:

- Long feature lists
- Generic UI pans
- UI -> zoom -> fade -> feature -> fade
- Corporate transitions
- Cheesy music

Motion story:

```text
Noise
Thousands of particles
One particle glows
Radar locks
Graph grows
Receipts connect
Memory expands
Terminal wakes
URL
```

The sequence should tell a story instead of showing screens.

Goal:

Make viewers feel like they discovered secret infrastructure.
